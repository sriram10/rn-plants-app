# Application Flow Screens
  -> Welcome Screen (Action Button : Start) - Stack
     -> Home Screen (Drawer menu - default menu: Store Home) - Drawer
        -> Stores Home (banner and top 10 stores list) - Stack
           -> Stores List (click on store card)
              -> Store Details
        -> About us
        -> Contact us
